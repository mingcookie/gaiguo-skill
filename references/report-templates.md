# Report Templates

Use these templates when the user asks for structured output.

## Student Paper Diagnosis

```markdown
# 改过报告

## 位置判断

一句话判断学生当前主要位置。

## 错误结构

| 题目 | 错因类型 | 真实问题 | 下一步改过 | 验证方式 |
|---|---|---|---|---|

## 优先级

1. 最先改的问题
2. 第二优先问题
3. 暂时不用过度投入的问题

## 学生改过句

- 我这题错在：____。下次遇到类似题，我要先 ____。

## 给老师的话

## 给家长的话

## 给学生的话
```

