# Mistake Taxonomy

Use this taxonomy for wrong-question, homework, and exam-paper diagnosis.

## Knowledge Gap

The student lacks the concept, fact, formula, rule, or procedure.

Signals:

- Cannot state the rule.
- Uses unrelated knowledge.
- Leaves the item blank because they do not know where to start.
- Makes the same concept error across different forms.

Correction:

- Rebuild the concept.
- Use one base example.
- Verify with one near-transfer example.

## Transfer Gap

The student knows the base knowledge but cannot use it when the question changes.

Signals:

- Can do textbook examples but fails variants.
- Says "I know this, but did not recognize it."
- Chooses the wrong method for a familiar concept.

Correction:

- Compare base question and variant.
- Ask: "What stayed the same? What changed?"
- Generate 2-3 related transformed questions.

## Reading / Review Gap

The student misses or misreads conditions.

Signals:

- Correct method, wrong object.
- Ignores "not", "except", units, range, or hidden condition.
- Error disappears when the question is read aloud.

Correction:

- Mark conditions before solving.
- Restate the question in one sentence.
- Check whether answer matches the question asked.

## Process / Habit Gap

The student understands but loses points through process.

Signals:

- Calculation slip.
- Formatting or unit error.
- Skips intermediate step.
- Time allocation causes unfinished work.

Correction:

- Add a fixed checking routine.
- Write key steps, not just final answer.
- Practice under a small time box.

## Expression Gap

The student knows the idea but cannot express it clearly.

Signals:

- Oral explanation is better than written answer.
- Answer lacks proof, structure, or key terms.
- Writing is vague, incomplete, or disordered.

Correction:

- Provide answer skeleton.
- Require key terms.
- Practice "because... therefore..." chain.

## State Gap

Performance is affected by attention, anxiety, fatigue, or speed.

Signals:

- Errors cluster late in the paper.
- Easy questions are missed under pressure.
- Performance fluctuates strongly.

Correction:

- Identify timing and condition.
- Use shorter timed sets.
- Add pre-test stabilization routine.

## Difficulty Avoidance

The student stops before truly attempting.

Signals:

- Gives up when text is long.
- Blank answer despite having partial knowledge.
- Says "I cannot" before trying first step.

Correction:

- Require only first-step attempt.
- Break task into visible footholds.
- Reward entry into the problem, not only correctness.

## Strategy Gap

The student uses poor sequencing or method selection.

Signals:

- Spends too long on low-value tasks.
- Uses brute force when a simpler method exists.
- Fails to choose between methods.

Correction:

- Teach decision rules.
- Build "when to use what" table.
- Practice method selection separately from calculation.
