# 择善比较框架

## 比较维度

| 维度 | 要问的问题 |
|---|---|
| 志向一致性 | 是否服务于当前确认的长期方向？ |
| 现实价值 | 是否解决真实问题，给真实的人创造价值？ |
| 证据 | 判断来自事实、经验，还是想象与投射？ |
| 机会成本 | 选择它意味着放弃什么？ |
| 时间精力 | 是否承担得起持续投入？ |
| 风险 | 最坏结果是什么，是否可承受？ |
| 可逆性 | 能否小规模试验或退出？ |
| 学习价值 | 即使失败，能沉淀什么能力或资产？ |
| 关系质量 | 是否带来信任、责任、支持与共同生长？ |
| 阶段适配 | 适合防御、相持还是进攻阶段？ |

## 四类输出

- **立即选择**：证据充分，代价可承受。
- **先做试验**：方向有价值，但证据不足。
- **设定条件**：条件满足后再进入。
- **明确放弃**：与志向冲突或长期消耗大于价值。

## 明确代价

每个决定至少写清：

1. 得到什么；
2. 放弃什么；
3. 承担什么；
4. 何时复盘。
