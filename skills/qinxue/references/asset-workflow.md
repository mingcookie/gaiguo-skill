# 数字资产工作流

## 基本链条

```text
经历 -> 材料 -> 照见自己 -> 生成洞见与能力 -> 作品 -> 资产 -> 再行动
```

## 最小归档字段

- 日期
- 来源
- 主题
- 客观内容
- 主观感受
- 关键问题
- 新增洞见
- 自身发生的变化
- 可验证行动
- 关联笔记

## 常见产物

| 输入 | 最小产物 |
|---|---|
| 一本书或一篇文章 | 原文单元、问题、个人理解、现实连接 |
| 一次聊天 | 事件摘要、对方观点、自己的问题、待验证判断 |
| 一次项目实践 | 目标、行动、反馈、差异、下一步 |
| 一段口述 | 带日期的日记或主题笔记 |
| 多次重复记录 | 模式、变化、例外、发展趋势 |

## 学有修无

把知识作为参照，不把知识当作身份。学习不是把别人压缩进自己的资料库，而是借由与知识相遇，改变自己的投影片，长出新的感受、判断、能力与创造。
