# 六大天问

## 标准结构

| 问题 | 盘点范围 |
|---|---|
| 我是谁 | 名号；兴趣、爱好、优势、劣势、个性、核心特质 |
| 我从哪里来 | 成长经历、关键转折、事故、黑历史、获得的经验 |
| 我要到哪里去 | 使命、愿景、事业、阶段目标 |
| 我有什么 | 知识、技能、经验、人脉、资产、资金 |
| 我要什么 | 身体健康、工作事业、收入金钱、资产资源、家庭幸福、时间空间 |
| 我能放弃什么 | 名誉、面子、关系期待、消费欲望、身份幻想、城市执着、短期安全感 |

## 分析原则

### 我是谁

区分自我描述与现实证据。不要把一时状态写成固定人格。

### 我从哪里来

整理事件、感受、应对方式和长期影响。不要擅自诊断创伤。

### 我要到哪里去

区分：

- 使命：愿意长期回应什么问题；
- 愿景：希望世界或生活变成什么样；
- 事业：通过什么现实载体持续行动；
- 阶段目标：当前一年或当前战役要完成什么。

### 我有什么

只盘点能够说明来源、使用场景或成果的资产。潜在人脉不等于已可调用资源。

### 我要什么

允许用户描述理想的一天，再反推时间、关系、空间、工作与收入条件。

### 我能放弃什么

取舍必须连接志向。放弃不是自我惩罚，而是守护主要方向。
