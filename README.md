# 改过 Skill

从错题到人生复盘，把错误变成位置、路径和下一步行动。

> 不表演勤奋，只确认学会。

## What It Does

`改过 Skill` helps an AI agent diagnose mistakes and generate correction paths.

It can be used for:

- Student wrong-question review
- Exam-paper diagnosis
- Teacher feedback reports
- Parent communication
- Personal or project retrospectives

## Core Idea

Errors are not proof of failure. Errors are coordinates.

The workflow is:

```text
发现错误 -> 确认位置 -> 分析原因 -> 生成改法 -> 复盘验证
```

## First Use Case

The first practical entrance is education:

```text
question + standard answer + student answer -> 改过报告
```
