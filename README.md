# 立志 · 勤学 · 改过 · 择善 Skills

> v0.1 开放原型：先把四个位置建立起来，再通过真实使用持续修订。

这是一组面向 AI 时代生命实践与教育实践的 Skill：

```text
立志 -> 勤学 -> 改过 -> 择善 -> 再立志
```

- **立志**：通过六大天问盘点生命材料，确认方向与现实起点。
- **勤学**：把阅读、经历、问题和实践转化为能力与数字资产。
- **改过**：把错误转化为位置、原因、行动与验成。
- **择善**：比较道路、机会、关系和代价，完成取舍与聚焦。

它们不是四套互不相干的方法，而是同一个生命生成循环的四个环节。

## 改过 Skill

从错题到人生复盘，把错误变成位置、路径和下一步行动。

> 不表演勤奋，只确认学会。

## Why This Exists

Most wrong-question systems help students collect mistakes.

`改过 Skill` asks a different question:

> Did the student actually learn from the mistake?

The visible pain is simple:

- Students do many exercises, but review is not targeted.
- Wrong-question notebooks often become a performance of effort.
- Teachers do not have enough time to diagnose every student's real position.
- Parents see scores, but not where the child is actually stuck.

The deeper idea is:

> Errors are not proof of failure. Errors are coordinates.

This repository keeps the eye-catching problem in the foreground, while preserving the deeper educational philosophy in the background:

- Foreground: wrong questions, exam diagnosis, targeted review, teacher efficiency.
- Depth: agency, growth, real learning, and helping people live more like human beings with AI.

## What It Does

`改过 Skill` helps an AI agent diagnose mistakes and generate correction paths.

It can be used for:

- Student wrong-question review
- Exam-paper diagnosis
- Teacher feedback reports
- Parent communication
- Personal or project retrospectives

## Core Idea

Errors are not proof of failure. Errors are coordinates.

The workflow is:

```text
发现错误 -> 确认位置 -> 分析原因 -> 生成改法 -> 复盘验证
```
